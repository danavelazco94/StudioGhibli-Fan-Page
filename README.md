# 🌿 Studio Ghibli Fan Page

Sitio web inspirado en el mágico universo de Studio Ghibli y sus películas más icónicas.  
El proyecto fue desarrollado como trabajo final de Desarrollo Web utilizando HTML, CSS, Sass y Bootstrap.

---

## 📖 Descripción

Studio Ghibli Fan Page es una fan page dedicada a compartir la magia, la fantasía y los valores presentes en las películas creadas por Hayao Miyazaki y Studio Ghibli.

El sitio incluye información sobre películas emblemáticas, personajes inolvidables, una galería visual inspirada en el estudio y una sección de contacto interactiva.

Además de buscar un diseño visual atractivo, el proyecto fue desarrollado teniendo en cuenta principios de responsive design, accesibilidad básica y organización semántica del contenido.

---

## 🎯 Objetivos del proyecto

- Aplicar conocimientos de HTML5 y CSS3
- Utilizar Sass para organizar estilos
- Implementar diseño responsive
- Incorporar componentes de Bootstrap
- Mejorar la estructura semántica y accesibilidad
- Crear una experiencia visual inspirada en la estética Ghibli
